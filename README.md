# Student-Form-Task-Js
Student Form By Using Html,Tailwind CSS and Basic Js
A simple and responsive student registration form built with HTML, Tailwind CSS, and vanilla JavaScript.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Form Validation** - Client-side validation for all input fields
- **Modern UI** - Clean and modern interface using Tailwind CSS
- **Interactive** - Real-time form interactions with JavaScript
- **Lightweight** - No heavy frameworks or dependencies

## Technologies Used

- **HTML5** - Semantic markup structure
- **Tailwind CSS** - Utility-first CSS framework for styling
- **JavaScript** - Form validation and interactivity

## Form Fields

The form includes the following fields:
- Full Name
- Email Address
- Date of Birth
- Gender
- Course/Program

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/student-form-task.git
```

2. Navigate to the project directory:
```bash
cd student-form-task
```

3. Open `index.html` in your web browser:
```bash
open index.html
```
Or simply double-click the `index.html` file.

## Usage

1. Fill out all required fields in the form
2. The form will validate your input in real-time
3. Click the submit button to submit the form
4. Form data can be processed or stored as per your requirements

## Customization

### Tailwind CSS
This project uses Tailwind CSS via CDN. If you want to customize the design:
- Modify the Tailwind classes in `index.html`
- Or set up Tailwind CSS locally for custom configurations

### JavaScript
Edit `script.js` to modify:
- Form validation rules
- Submit behavior
- Additional interactive features
